# K-Means Clustering

A beginner-friendly implementation of the **K-Means Clustering** algorithm using **Python** and **Scikit-learn** to group similar data points and identify hidden patterns in data.

# 🚀 Key Features
- Data preprocessing
- Feature scaling
- Elbow Method (WCSS)
- K-Means clustering
- Cluster visualization
- Silhouette Score evaluation

🛠️ Tech Stack
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

📌 Machine Learning Skills
Unsupervised Learning • Clustering • Data Preprocessing • Feature Scaling • Data Visualization • Model Evaluation • WCSS • Elbow Method • Silhouette Score
